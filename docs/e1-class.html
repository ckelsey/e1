e1-class

Evaluates the supplied value and adds/removes classes to the element. There are 2 ways to go about it:

  • You can supply a single or comma separated string of bindings that will add their values as classes.
  • You can supply a single or comma separated string of bindings, each with a conditional binding and a value binding separated by a colon. If the conditional binding has a value, the value binding will be added as a class.
